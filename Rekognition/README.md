## 💰 Pricing

- 🖼️ **Image Analysis**: $1.00 per 1,000 images
- 🎬 **Video Analysis**: $0.10 per minute
- 👤 **Face Recognition**: $1.00 per 1,000 images
- 📝 **Text Detection**: $1.50 per 1,000 images
